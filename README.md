# Finance-Domain-Analysis-in-Python

This project focuses on performing Finance Domain Data Analysis using Python. It includes data ingestion, cleaning, exploratory analysis, and validation tasks such as accurately determining row counts in large financial datasets. The project aims to improve data quality, identify inconsistencies, and prepare reliable datasets for reporting and decision-making in financial operations.
